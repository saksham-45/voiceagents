# Demo Flow

## Setup

- start LMMS with the AgentControl plugin installed
- open `Tools -> Agent Control`
- keep a short audio file available for import

## Suggested demo sequence

1. `add 808`
2. `show mixer`
3. `new instrument kicker`
4. `open slicer`
5. `add effect amplifier`
6. `import audio /full/path/to/file.wav`

## Demo point

The key claim is not that every command is hardcoded. The key claim is that LMMS exposes a stable command execution boundary now, and the text and voice agents can both target it locally.
